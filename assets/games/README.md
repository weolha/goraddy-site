# 게임 에셋 양식 (회사 사이트용)

회사 사이트(루트 index/games)는 게임마다 아래 에셋을 씁니다. 폴더에 같은 파일명으로 넣으면 자동 반영돼요.

```
assets/games/<게임슬러그>/
├─ icon.png       앱 아이콘 — 정사각형, 1024×1024 (배경 채움, 모서리 둥글림은 CSS가 처리하니 각진 원본)
├─ feature.webp   대표 가로 키아트 — 16:9 (권장 1600×900), 게임의 메인 일러스트
├─ tile.webp      타일용 키아트 — 정사각/세로 위주 크롭 (권장 1000×1000~1200), 미니 타일 배경
└─ shot1~3.webp   인게임 스크린샷 — 세로 캡처 (권장 높이 1000~1400px), 3장 권장
```

슬러그: `goraddy`, `ponpondoodoo2`, `ponpondoodoo`, `aim`

## 현재 상태 (2026-06-13)

- **goraddy** — ✅ icon + feature + tile + shot1~3
- **ponpondoodoo2** — ✅ icon + feature(키아트)
- **ponpondoodoo** — ✅ icon + feature(키아트)
- **aim** — ✅ icon + feature(가로 프로모)

→ 게임 페이지 4종 모두 가로 키아트 피처 배너로 노출 중. 피처 이미지는 **가로면 비율 상관없이 원본 그대로 꽉 차게** 표시(크롭 없음).

추가하면 더 좋은 것(선택): 각 게임 `shot1~3.webp`(세로 스크린샷) — 넣으면 고래디처럼 피처 아래 스크린샷 스트립을 깔아드려요. 세로 게임은 세로 스크린샷이 스트립에 잘 어울립니다.

## 형식 팁

- webp 권장(가볍고 품질 좋음). png/jpg도 가능 — 넣을 때 파일 확장자에 맞게 HTML의 src만 바꾸면 됨.
- 가로 키아트는 인물/로고가 가운데~위쪽에 오도록(상단 크롭 대비). 
- 스크린샷은 동일한 높이 비율로 맞추면 스트립이 깔끔함.
